# ⚔️ タイピング忍者

忍者をテーマにしたタイピングゲームです。画面を流れる単語を素早く入力して撃退しましょう！

## 遊び方

1. `index.html` をブラウザで開く
2. 難易度を選ぶ（かんたん / ふつう / むずかしい）
3. 「はじめる」をクリック
4. 画面を流れる単語を入力してEnterを押さずにタイプ
5. 単語が画面外に消えるとライフが1減る（ライフは3）

## ルール

- 単語を正確にタイプすると斬撃エフェクトとスコアが入る
- 連続して倒すとコンボボーナスが加算される
- 8語倒すごとにレベルアップ（速度が上がる）
- ライフが0になるとゲームオーバー

## 難易度

| 難易度 | 単語 | 速度 |
|--------|------|------|
| かんたん | 英単語（短め） | 遅い |
| ふつう | 英単語（中級） | 普通 |
| むずかしい | プログラミング用語 | 速い |
